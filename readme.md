# Couple Choco

## 這是什麼？

這是串接綠界金流站內付 2.0 的範例網站，分別以 React 與 Node.js 實作前後端，並以 Tailwind CSS 切版，另使用 Swiper.JS 與 AOS.JS 製作特效。商品圖片與文案以 Claude AI 與 MidJourney 製成。
